Robotics Summer Projects

Hey Y24s! Robotics Club brings you exiciting summer projects, which would help you build hands-on-skills, along with experience in robotics.

AUTONOMOUS NAVIGATION

This project focuses on developing a fully autonomous mobile robot using the TortoiseBot platform. The objective is to enable the robot to explore an unknown environment, create a real time-time map using SLAM, and navigate efficiently to user-defined goals using path planning algorithms within the ROS framework.

BALL BALANCING ROBOT

Here, we build a robotic system that can balance a ball on a flat platform using two servo motors and a Rasberry Pi. A camera mounted above the platform will track the ball's position in real time using CV, and a PID control algorithm will adjust the platform angle to keep the ball centered. By the end, we will have a robot that can balance object easily.

HEXAPOD

This project focuses on designing and testing the control and locomotion system of a hexapod robot. The robot utilizes and= RC-based communication system to execute smooth walking and obstacle-climbing maneuvers. Various gait pattern are implemented for efficient movement and enanced stability on uneven terrain. The ultimate goal is to develop a terrain-adaptive hexapod suitable for real-world application and research in robotics and mobitlity.

MECHANICAL TURK

In this project, we will build a robot which would play chess. So if you ever feel that playing chess over a board is far more satisfying than playing online, come pay the Turk a visit!

STEP XPLORER

In this project, we will build a stair climbing robot. It can detect and climb stairs on its own using ultrasonic sensors. The robot will also have a strong and stable platform to carry things while moving. It will stay balanced even when going over rough or uneven surfaces.
